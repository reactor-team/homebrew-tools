# Reactor Homebrew Tap

This tap distributes official CLI tools from [Reactor Technologies](https://reactor.inc).

## Usage

```bash
brew tap reactor-team/tools
brew install reactor-cli
```

Or in one command:

```bash
brew install reactor-team/tools/reactor-cli
```

## Available Formulae

| Formula       | Description                                             |
| :------------ | :------------------------------------------------------ |
| `reactor-cli` | Partner CLI for uploading images and model weights      |

## Formula Management

Formulae in this tap are automatically updated by the Reactor CI/CD pipeline
on every release. Do not edit formula files by hand.
